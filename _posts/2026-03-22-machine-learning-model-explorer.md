# 🧠 What This Tool Does

- 📂 Upload your dataset (CSV)
- 🔍 Perform Exploratory Data Analysis (EDA)
- ⚙️ Apply feature engineering & preprocessing
- 🤖 Train multiple ML models
- 📊 Evaluate performance (ROC, AUC, F1, Confusion Matrix)
- 🔮 Generate predictions on new data

---

# ⚡ Key Features

- 💻 100% Browser-Based (no setup)
- 🧩 Step-by-step guided workflow
- 📊 Built-in visualizations
- 🧠 Multiple ML algorithms
- 📈 Real-time evaluation

# 🪜 Step-by-Step Usage

## 📥 1. Load Dataset
- Upload CSV
- Ensure binary target column
- Index columns auto-removed

## 🎯 2. Select Target & Features
- Choose binary target
- Define positive class
- Configure feature types

## 🔍 3. Explore Data
- Summary stats
- Class balance
- Visualizations

## ⚙️ 4. Preprocess Features
- Apply scaling / transformations

## 🎛️ 5. Configure Models
- Select models
- Adjust split

## 🏋️ 6. Train
- Run pipeline
- Monitor progress

## 📊 7. Results
- Metrics + visualizations

## 🔮 8. Predict
- Upload new data
- Generate predictions
